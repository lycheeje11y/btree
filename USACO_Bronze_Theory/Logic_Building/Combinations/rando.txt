958. Sum Target
268
Input Format
The first line of the input contains N and K. N denotes the number of integers on the next line. K indicates the target sum.
The second line of the input contains N unique integers separated by spaces.
Output Format
Print all of the 3 number combinations that sum to K in ascending order.
Sample Input

6 10
5 4 6 2 1 3

Sample Output

1 3 6
1 4 5
2 3 5
